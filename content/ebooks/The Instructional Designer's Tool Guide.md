tagline: A hands-on, step-by-step handbook that walks instructional designers
  through the exact tools of the trade — from storyboarding to Articulate,
  Figma, Canva, Captivate, and launching a course on an LMS.
summary: This is a practical, tool-by-tool handbook for instructional designers
  covering the full ADDIE workflow (Analyze, Design, Develop, Implement,
  Evaluate) alongside step-by-step walkthroughs of the industry's core software.
  It opens with instructional design fundamentals and storyboarding best
  practices, then guides readers through Articulate Storyline 360 and Rise 360,
  Figma for e-learning layouts and prototypes, Canva for infographics and job
  aids, Adobe Captivate for simulations and quizzes, and Google Workspace for
  SME collaboration and needs analysis. It covers major LMS platforms and SCORM
  packaging, accessibility standards for compliant course design, a curated list
  of communities and resources, and closes with a final project checklist to
  launch a polished course.
problem: New and practicing instructional designers are often taught the theory
  of course design without ever being shown, click by click, how to actually use
  the tools of the profession — leaving them able to plan a course but unsure
  how to build one in Storyline, prototype it in Figma, or package and upload it
  to an LMS. Tutorials for each tool exist in isolation, scattered across vendor
  docs and forums, with no single resource connecting the fundamentals of
  instructional design to the concrete software workflow needed to produce a
  finished, accessible, launch-ready course. This handbook closes that gap by
  pairing ID theory with guided, step-by-step tool walkthroughs in one
  reference.
takeaways:
  - The ADDIE framework (Analyze, Design, Develop, Implement, Evaluate) and how
    to apply it when planning a course
  - How to create a course outline, choose a storyboard format, and write
    content before designing visuals
  - How to build a course in Articulate Storyline 360, including scenes,
    triggers, states, layers, and quizzes
  - How to structure and publish a course in Articulate Rise 360 using sections,
    lessons, content blocks, and knowledge checks
  - "How to use Figma for e-learning: frames, grids, reusable components,
    styles, clickable prototypes, and exporting assets"
  - How to design infographics, job aids, and quick reference cards in Canva
    using a Brand Kit
  - How to build software simulations, interactions, and quizzes in Adobe
    Captivate and publish them for an LMS
  - How to use Google Workspace to collaborate with SMEs, track drafts with
    Version History, and build needs-analysis surveys
  - How major LMS platforms (Moodle, Canvas, TalentLMS, Blackboard) differ, and
    how to package, upload, and track a course with SCORM
  - How to apply accessibility standards — color contrast, alt text, closed
    captions, keyboard navigation, and screen reader testing
  - How to use a final project checklist covering content, technical setup,
    accessibility, and launch readiness
