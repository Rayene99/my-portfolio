tagline: An interactive audit dashboard that helps departments track, score, and
  improve the accessibility of every course in a pilot program.
summary: An interactive dashboard built for a department-wide accessibility
  pilot program, combining a live course audit checklist, accessible content
  templates, and a full accessibility guidelines reference. It tracks courses
  enrolled, fully audited courses, flagged issues, and an overall department
  accessibility score, alongside guidance on POUR principles, WCAG AA
  conformance, heading structure, alt text, PDF accessibility, captions,
  transcripts, contrast ratios, and Universal Design for Learning (UDL), with
  generated department reports.
problem: Departments running multiple courses often lack a consistent way to
  track accessibility compliance across them, leaving instructors to interpret
  WCAG and UDL guidelines on their own with no shared checklist, no visibility
  into which courses have been reviewed, and no easy way to report progress to
  leadership.
approach: Built a single interactive dashboard combining a structured audit
  checklist (with practical tips per item), ready-to-use accessible templates,
  and a condensed accessibility guidelines reference covering POUR, WCAG AA
  conformance, and UDL principles, so instructors can audit a course and
  immediately see what to fix.
outcome: Departments can see real-time stats — courses enrolled, fully audited
  courses, issues flagged, and an overall accessibility score — and generate a
  department report summarizing accessibility progress across the entire pilot
  program.
